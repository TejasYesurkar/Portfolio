Overview

In This Application salesman take an Order From Local Retailer Shopes. That Order display to the distributor and dispatch the order from Distributor.

In order to help distributor through this process, can easy take an Order From Local Shopes And Salesman also Track the Retailer shop By showing route from its recent location.

With the help of Cordova we can convert Web Application into Mobile Application for ios and andoird users.
